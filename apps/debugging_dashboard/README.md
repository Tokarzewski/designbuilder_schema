# Debugging App 

## workflow
1. Upload dsbXML file to app
2. Select checkboxes to activate scripts that should be run
3. Generate reports 

## scripts
- which fields are undefined?
- problematic geometry
- model size (nr of zones, surfaces, openings, hvac components)

## libraries
GUI - Dash or Streamlit  
Plots - Plotly  
Graph - NetworkX  
Query - desingbuilder_schema
